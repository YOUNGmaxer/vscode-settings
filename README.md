# vscode-settings

收录 VSCode 前端开发配置，提升开发体验和项目规范。

**目标**

* 针对常用配置，给出推荐性的配置

* 提供各种常用场景的配置推荐

## 场景

### 代码自动格式化

> 参考：[format.json](./settings/format.json)

支持格式化以下类型文件：

|文件类型|依赖插件|
|-|-|
|json|vscode.json-language-features|
|ts|esbenp.prettier-vscode|
|vue|Vue.volar|
|yaml|redhat.vscode-yaml|
|shell|foxundermoon.shell-format|
